# Omnitranslate.

omnitranslate is just a tui tool i made to look like a hacker, dw its harmless.
## Installation: Source and Downloads.

You can clone this repo using ```git clone https://github.com/UIFreestyler/omnitranslate```
Easy. Now: cd omni* (* to not need to type everything)
Building:
make clean
make install
make
You already builded it! Now just execute it!
[sudo]chmod +x omnitranslate
./omnitranslate
Thats all friends! I hope yall enjoy!
### Note:
This converts various things, but not speaking languages. For that just use Google Translate API or go to [Google Translate](translate.google.com)
